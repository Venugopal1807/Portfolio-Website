Case Study: Fuel Logistics Platform (Zapygo)

Built and shipped a real-time fuel logistics platform with 3 apps (Customer, Driver, Admin) handling secure doorstep fuel delivery. The system required solving real-world challenges like low network environments, fuel theft prevention, and multi-role coordination.

Implemented BLE-based fuel validation (50m + OTP), geo-based service zoning, and a full order lifecycle from request to delivery. Designed offline-first architecture using SQLite/Room with sync logic to ensure uninterrupted operations in remote areas.

Worked on a shared database system with separate backend services to maintain stability and reduce coupling. Integrated HDFC SmartGateway payments and enabled real-time tracking for operational transparency.

Outcome: Production-grade system supporting real-world delivery workflows with secure dispensing, offline reliability, and multi-app coordination.
